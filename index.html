Personal Expense Manager

This application is created for object (-) oriented programming and using CRUD operation.

In this application, Frontend is using Windows Form.

1.Home

1. In this Form you see a list of Expenses and Add Expenses, Edit Expenses, View Expenses, Remove Expenses, and the total of Existing Expenses.

2. When this Form loads get the Existing Expense Category from Databases to show a list of Expenses to Listbox.

3. If you Add New Expense you Click the Add button you can add a new Expenses.

4. If you view Existing Expenses first select the category, then click the View Expenses button.

5. If you edit Existing Expenses first select the category, then click the Edit Expenses button.

6. If you Remove Existing Expenses first select the category, then click the Remove Expenses button.

2.Add Expense

1. In this form you see an existing list of expenses and total expense amount.

2. If you add a new expense click the add button.

1. First select the category for Combobox.

2. Then give a heading for the textbox.

3. Then give an amount for the textbox.

4. Then select a date for the date and time picker.

3.Edit Expense

1. In this form you edit existing expenses and first select the existing category.

2. Then click the Edit Expenses button.

1. In this form you see existing expense headings and amounts for a particular category.

2. If you Edit a particular expense, First select the heading.

3. Then click the Edit Expense button.

1.In this form you see Category, Heading, Expense, and Date.

2.In this form you change only the Expense and Date.

3.Then you click the save button

4.View Expense

1.In this form, you select an existing category.

2.Then click the View Expenses button.

1. In this form you see existing expense headings and amounts for a particular category.

2. If you view a particular expense, First select the heading.

3. Then click the View Expense button.

1. In this form you see Category, Heading, Expense, and Date.

2. Then you Click the Exit button back to the home form.

5.Remove Expense

1. In this form you select an existing category.

2. Then click the Remove Expenses button.

1. In this form you see existing expense headings and amounts for a particular category.

2. If you remove a particular expense, First select the heading.

3. Then click the Remove Expense button.

Classes

Expense class

1. In this class have Expense properties for Category, Heading, Amount, Date

Expenses class

1. In this class have a list of expenses.

Expense Repositry class
