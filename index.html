Personal Expense Manager

In this application develop to the object oriented programing and CRUD operation.

In this application Frontend is using Windows Form.

1.Home

1.In this Form you see list of Expenses and Add Expenses,Edit Expenses,View Expenses,Remove Expenses and the total of Existing Expense.

2.When this Form load get the Existing Expense Category from Data base to show list of Expenses to listbox.

3.If you Add Expense you Click the Add button you can add new Expense.

4.If you view Existing Expense first select the category, then click the View button.

5.If you edit Existing Expense first select the category, then click the Edit button.

6.If you Remove Existing Expense first select the category, then click the Remove button.

2.Add Expense

1.In this form you see existing list of expenses and total expense amount.

2.If you add new expense click the add button.

1.First select the category for combobox.

2.Then give the heading for textbox.

3.Then give the amount for textbox.

4.Then select the date for date and timepicker.

3.Edit Expense

1.In this form you edit existing expense first select the existing category.

2.Then click the Edit Expenses button.

1.In this form you see existing expense heading and amount for particular category.

2.If you Edit particular expense,First select the heading.

3.Then click the Edit Expense button.

1.In this form you see Catagory,Heading,Expense,Date.

2.In this form you change only Expense and Date .

3.Then you click the save button

4.View Expense

1.In this form you select existing category.

2.Then click the View Expenses button.

1.In this form you see existing expense heading and amount for particular category.

2.If you view particular expense,First select the heading.

3.Then click the View Expense button.

1.In this form you see Catagory,Heading,Expense,Date.

2.Then you Click the Exit button back to home form.

5.Remove Expense

1.In this form you select existing category.

2.Then click the Remove Expenses button.

1.In this form you see existing expense heading and amount for particular category.

2.If you remove particular expense,First select the heading.

3.Then click the Remove Expense button.
